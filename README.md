# 中小学教务系统

#### Description
该系统是我经过2个月的时间开发的，当然功能也不是特别的好，但是可以学习。
该系统使用的是springboot+ mybatis-plus + layui + shiro架构。
该系统包括了教师信息管理、学生信息管理、教室信息管理、考试计划管理、考试成绩管理、学生成绩统计、智能排课系统（不是特别的健全，但是基本的够了，不排除还有bug）、课表查询、考务查询等等功能。
